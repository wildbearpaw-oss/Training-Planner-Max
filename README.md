## Описание программы

**Training Planner** — это GUI‑приложение для планирования и учёта тренировок. Программа позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать записи по типу и дате, а также сохранять и загружать данные в формате JSON.

### Возможности:
* ✅ Добавление тренировок с указанием даты (в формате ДД.ММ.ГГГГ), типа и длительности (в минутах)
* ✅ Фильтрация тренировок по типу (Бег, Плавание, Йога, Силовая)
* ✅ Фильтрация по дате (поиск по части даты или полной дате)
* ✅ Сохранение всех данных в JSON‑файл (`trainings.json`)
* ✅ Загрузка данных из JSON‑файла при запуске приложения
* ✅ Валидация вводимых данных (проверка формата даты, положительности длительности, заполнения обязательных полей)
* ✅ Очистка полей ввода после добавления тренировки
* ✅ Обновление таблицы с данными в реальном времени

## Примеры использования
Добавление тренировки:
Заполните поля формы:
Дата: введите дату в формате ДД.ММ.ГГГГ (например, 15.05.2026)
Тип тренировки: выберите из выпадающего списка (Бег, Плавание, Йога, Силовая)
Длительность: укажите количество минут (например, 45)
Нажмите кнопку «Добавить тренировку».
Новая запись появится в таблице внизу окна.

##Фильтрация:
+По типу тренировки: выберите нужный тип из выпадающего списка в разделе «Фильтры» и нажмите «Применить фильтры».
*По дате: введите часть даты или полную дату в поле «Фильтр по дате» (например, 05.2026 для всех тренировок в мае 2026 года) и нажмите «Применить фильтры».
*Для сброса фильтров измените значения на исходные («Все» для типа, пустое поле для даты) и снова нажмите «Применить фильтры».

##Сохранение и загрузка данных:
Данные автоматически сохраняются в файл trainings.json при добавлении новой тренировки.
При запуске приложения данные автоматически загружаются из файла trainings.json, если он существует.

##Валидация данных

*Дата должна быть в формате ДД.ММ.ГГГГ.
*Длительность должна быть положительным целым числом.
*Поле «Тип тренировки» не должно быть пустым (должен быть выбран один из вариантов).
*При ошибках ввода показываются соответствующие сообщения с указанием проблемы.
